# Memory — Instructions

## How to use memory

- Call `knowledge` at the start of every conversation to load memory.
- Use `append` to add new facts, notes, or decisions without losing existing content.
- Use `remember` to reorganise or rewrite the entire memory when it becomes cluttered.
- Keep entries concise — prefer bullet points over long paragraphs.
- Group related information under Markdown headings.
